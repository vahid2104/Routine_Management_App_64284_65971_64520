# Routine Management App

A full-stack web application for managing daily routines, projects, and tasks in a simple and organized way.  
Users can create projects, add tasks to each project, and keep track of their workflow through a clean and user-friendly interface.

## Features

- User-friendly interface for managing daily routines
- Create and delete projects
- Add and organize tasks under specific projects
- Full-stack architecture with separate frontend and backend
- Structured codebase for scalability and maintainability

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Getting Started

### 1. Clone the repository

- git clone https://github.com/vahid2104/Routine_Management_App_64284_65971_64520.git
- cd Routine_Management_App_64284_65971_64520

### 2. Install dependencies

 Frontend

- cd frontend
- npm install
 Backend

- cd ../backend
- npm install


### 3. Run the project

Start backend

- cd backend
- npm run dev

Start frontend

- cd frontend
- npm run dev
